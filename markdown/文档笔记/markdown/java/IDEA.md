####IDEA快捷操作

一、注意：如果已经写好代码块，使用块注释需要先选中要注释的块；

 使用**Ctrl+ /**， 添加行注释，再次使用，去掉行注释

 使用 **Ctrl+Shift+ /**， 添加块注释，再次使用，去掉块注释；

 输入/** ,点击“Enter”，自动根据参数和返回值生成注释模板
